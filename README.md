# 💪 筋トレ記録アプリ with アスカ・ラングレー

エヴァンゲリオンのアスカ・ラングレーがあなたのトレーニングを応援＆指導する筋トレ記録アプリです！

## ✨ 特徴

### 👧 アスカのボイスシステム
- **オーバーロード成功時**: アスカが強気で褒めてくれます（ツンデレ）
- **オーバーロード失敗時**: アスカが心配しながらも励ましてくれます
- ランダムで10通りのセリフ

### 📊 主な機能

1. **トレーニング記録**
   - 重量、回数、セット数を記録
   - 17種類の種目を収録（胸、背中、脚、肩、腕）

2. **前回セット情報の自動表示**
   - 種目を選択すると前回の記録が自動表示
   - 入力フォームに自動入力されるので調整が簡単

3. **科学的オーバーロード提案**
   - 種目ごとに最適化された負荷増加パターン
   - 重量、回数、セット数の複数の提案（6〜7パターン）
   - より細かい重量調整（1kg、1.25kg、2.5kg、5kg、10kg刻み）

4. **データ可視化**
   - 総負荷量の推移グラフ
   - 種目別最大重量のグラフ

5. **履歴管理**
   - トレーニング部位ごとにグループ化
   - 時系列で記録を表示

6. **データエクスポート**
   - マークダウン形式でエクスポート
   - Obsidianなどのノートアプリで活用可能

## 🚀 使い方

1. `workout_app_updated.html` をブラウザで開く
2. 種目を選択
3. 重量・回数・セット数を入力（前回の記録が自動表示されます）
4. 「記録する」ボタンをクリック
5. アスカのコメントとオーバーロード提案を確認！

## 💾 データ保存

- データはブラウザのLocalStorageに保存されます
- ブラウザのデータをクリアしない限り、記録は保持されます

## 🎨 カスタマイズ

### アスカのセリフを変更する

`asukaComments` オブジェクトを編集してください：

```javascript
const asukaComments = {
    success: [
        "やるじゃない！前回より重量上がってるわよ！...",
        // ここに新しいセリフを追加
    ],
    failure: [
        "ちょっと！何やってんのよ！...",
        // ここに新しいセリフを追加
    ]
};
```

### 種目を追加する

`exerciseStrategies` オブジェクトに新しい種目を追加：

```javascript
'新しい種目名': {
    type: 'compound', // または 'isolation'
    repRange: '8-12',
    description: '種目の説明',
    strategy: '重量を優先的に増やす',
    weightIncrements: [2.5, 1.25, 5],
    tips: 'フォームのコツ',
    bodyPart: '胸' // または '背中', '脚', '肩', '腕'
}
```

そして、HTMLの `<select>` 要素に追加：

```html
<option value="新しい種目名">新しい種目名</option>
```

## ⚠️ 注意事項

- このアプリはローカルで動作します（インターネット接続不要、Chart.js以外）
- データはブラウザに保存されるため、別のブラウザでは共有されません
- データのバックアップは定期的にエクスポート機能を使用してください
